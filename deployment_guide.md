\# Deployment Guide



\## Project: Mental Health Status Classification from Social Media Text



\## Live App

https://6yfyvnqza8mhswzv3jy97t.streamlit.app/



\## Local Setup



\### Step 1 - Clone the repository

git clone https://github.com/hibamadathil-cmd/mental-health-status-classification-from-social-media-text-predictive-project-3.git



\### Step 2 - Install dependencies

pip install -r requirements.txt



\### Step 3 - Run the app

streamlit run app.py



\## App Features

\- Input social media text

\- Predicts mental health category

\- Shows confidence scores

\- Displays LIME explanations

\- Word cloud visualization



\## Model Files

\- svm\_model.pkl — Trained SVM model

\- le.pkl — Label encoder

\- cleaned\_dataset.csv — Training data
